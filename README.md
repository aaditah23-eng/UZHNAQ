# UZHNAQ Static Site

This directory was reorganized from a mirrored site export into a simpler static-site structure.

## Main Pages

- `index.html` - home page
- `about.html` - company overview
- `products.html` - products page
- `industries.html` - industries served
- `machines.html` - machine suite
- `contact.html` - enquiry/contact page
- `404.html` - fallback not-found page

## Assets

- `assets/images/` - site images used by the HTML pages
- `assets/scripts/` - custom JavaScript helpers
- `assets/source/` - source media and brand materials for editing

## Notes

- Internal page links were updated to plain `.html` files for easier local editing and static hosting.
- Some remote-hosted fonts, videos, and remote assets still remain inside the page HTML.
